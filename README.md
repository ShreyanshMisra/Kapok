# Kapok

The National Center for Technology and Dispute Resolution is an organization that supports developing technology for conflict management. Kapok is an application that is being developed to help coordinate volunteers for disaster relief efforts. 

## Getting Started

FlutterFlow projects are built to run on the Flutter _stable_ release.

Steps for setup
1. Run the command `flutter upgrade` to make sure your version is up to date
2. Install dependencies using `flutter pub get`
3. You can start the frontend in the emulator of your choice by running the `main.dart` file in the `lib` directory
