# Kapok

Kapok is a mobile app (built with Flutter & Firebase) for coordinating volunteer disaster relief teams, 
developed for the National Center for Technology and Dispute Resolution.

## Structure
- `/app` – Flutter mobile app (iOS/Android)
- `/firebase` – Firebase configuration & Firestore rules
- `/docs` – Design, architecture, and planning docs

## Setup
1. `cd app`
2. `flutter pub get`
3. `flutter run`
